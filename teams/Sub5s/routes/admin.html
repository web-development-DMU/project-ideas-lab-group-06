Admin Routes

Overview

Admin routes for Caloryx provide access to administrative functionalities such as user management, food database management, analytics, and system settings. These routes are protected and require admin authentication to ensure that only authorized personnel can access sensitive operations and data.

Admin Dashboard

GET
/admin
Admin dashboard with management options and system overview

User Management

GET
/admin/users
List all users in the system
Features: Supports filtering by username, includes links to user management pages
GET
/admin/users/:userId
Edit or delete a specific user account
Shows: Forms for editing user details and deleting accounts

Food Database Management

GET
/admin/food-database
Manage food items in the database
Features: Lists all foods, form for adding new food items with nutritional information
GET
/admin/food-database/:foodId
Edit or delete a specific food item
Functions: Update nutritional values, remove outdated or duplicate entries

Analytics & Reporting

GET
/admin/analytics
View application analytics and user statistics
Metrics: User growth trends, popular foods/exercises, system usage patterns, user engagement
GET
/admin/reports
Generate and view system reports
Includes: User activity reports, system health checks, data integrity reports

System Settings

GET
/admin/settings
Configure system-wide settings and preferences
Options: Daily intake defaults, macro ratio presets, maintenance mode, backup settings
