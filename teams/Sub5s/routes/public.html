Dashboard & Summary

GET
/dashboard
User dashboard displaying daily calorie summary
Shows: Total calories consumed, calories burned, net calories, progress toward daily goals
GET
/settings
User account settings and preferences
Configurable: Daily calorie goal, macronutrient targets, profile information

Meals Management

GET
/meals
List all meals for the logged-in user
Features: Supports filtering by date/name, includes links to individual meals
GET
/meals/:mealId
View details of a specific meal
Shows: meal name, calories, protein, carbs, fats, date/time eaten

Exercises Management

GET
/exercises
List all exercises for the logged-in user
Features: Supports filtering by date, includes links to individual exercises
GET
/exercises/:exerciseId
View details of a specific exercise
Shows: exercise name, calories burned, duration, date/time, intensity
